# MAGNITUDE

**The observable universe in one scrollbar.**

A single self-contained HTML page: a continuous WebGL zoom through 43 powers of ten,
from inside a proton (10⁻¹⁶ m) to the edge of the observable universe (8.8×10²⁶ m).
Every ~900 pixels of scroll multiplies the field of view by ten.

- 13 scenes, all hand-written fragment shaders — no images, no video, no libraries
- Live readout: current scale in meters, SI prefix, and the time light needs to cross your screen
- Real measured physics pinned at every landmark
- **Voyage** mode: hands-free cruise through the whole journey (~2.5 min)
- **Sound** mode: a generative drone whose pitch is your current order of magnitude
- Zero dependencies, zero network requests

## Run

Open `index.html`, or visit the deployed page.
